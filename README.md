## 프로젝트 환경
* 최소타켓: iOS 13.0
* iPhone: Portrait 지원
* 적용 디자인 패턴: MVC, Singleton, Repository
* 다크모드 지원
* 라이브러리: Realm, SnapKit

## 적용 기술
* SnapKit
* Realm CRUD
